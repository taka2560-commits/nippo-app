# 作業日報自動入力システム

現場作業者向けの、シンプルで使いやすい作業日報入力アプリです。
スマートフォンでの操作に最適化されており、PWA（Progressive Web App）としてホーム画面に追加して利用できます。

## 📱 主な機能

1. **簡単入力**: マスタ登録された選択肢から選ぶだけで、スムーズに日報を作成できます。
2. **自動保存**: 入力したデータはブラウザ内に自動的に保存されます（サーバー送信不要）。
3. **オフライン対応**: PWAとしてインストールすれば、電波の悪い場所でもアプリを開いて入力できます。
4. **月報エクスポート**: 毎月の締め日に合わせて、ExcelおよびPDF形式で月報を出力できます。
5. **柔軟なカスタマイズ**: 作業者名、現場名、作業内容などのリストを自由に編集・管理できます。

## 🚀 使い方

### 1. アプリのインストール（推奨）
スマートフォンでアクセスし、ブラウザのメニューから**「ホーム画面に追加」**を選択してください。
アイコンが追加され、アプリのように全画面で利用できるようになります。

### 2. 日報の入力
1. **日付選択**: デフォルトで当日が選択されています。過去の日付も選択可能です。
2. **作業者・現場選択**: 参加した作業者と現場を選択します。
3. **作業内容入力**: 作業内容、人工、時間（早出・残業）を入力します。
   - 「＋作業を追加」ボタンで複数の作業を登録できます。
   - 選択肢にない項目は直接入力も可能です。
4. **送信**: 「日報を送信する」ボタンを押すと保存されます。

### 3. 日報の修正・削除
1. カレンダーで過去の日付を選択します。
2. データが存在する場合、自動的に読み込まれ「再編集モード」になります。
3. 内容を修正して「日報を更新する」ボタンを押すか、削除ボタンでデータを消去できます。

### 4. 月報のエクスポート
フォーム下部の「月報エクスポート」エリアから操作します。
- **対象年月**: 出力したい年月を選択します（期間は前月16日〜当月15日）。
- **Excel出力**: 集計用データとして `.xlsx` ファイルをダウンロードします。
- **PDF出力**: 印刷・提出用の `.pdf` ファイルをダウンロードします。
  - A4横1ページに収まるようにフォーマットされています。

### 5. 各種設定（マスタ管理）
画面右上の⚙️アイコンから設定画面を開けます。
- **作業者リスト**: 作業者の追加・削除
- **現場リスト**: グループ（ゼネコン等）ごとの現場名管理
- **作業内容・材料**: よく使う項目の管理
- **設定のリセット**: 初期状態に戻すことも可能です。

## ⚠️ 注意事項
- データはご利用のブラウザ（端末）内に保存されます。
- ブラウザのキャッシュクリア（履歴の削除等）を行うと、過去の日報データが消える可能性がありますのでご注意ください。
- 機種変更時のデータ移行機能は現時点ではありません。必要なデータはこまめにExcel出力してバックアップすることをお勧めします。
